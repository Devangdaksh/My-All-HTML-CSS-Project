## ✨ Features

- 🖼️ Responsive layout
- 🛍️ Product showcase sections
- 🔍 Search bar
- 🧑 User account & cart icons

## 🚀 Getting Started

1. Clone or download this repository.
2. Open `Amazon/Amazon.html` in your browser.
